**Radio Free Airlift** — comprehensive overhaul of xian's personal web presence, much dating to the 1990s. Named after Radio Free Blogistan, xian's early-2000s blog (a cultural reference in the blogging/social software scene, connected to BloggerCon and early web culture). Before that: Enterzone (1994-1998 webzine).

**Four major pieces:**
1. **ezone.org + subsites** (Rackspace) — Enterzone webzine archive (1994-1998). Mostly static. Ambitious goal: curate a "digital coffee table book." Blocked on repair step via archive.org. Rackspace hosting is expensive for static content.
2. **mediajunkie.com** (WPEngine) — WordPress blog with rich archives. Needs migration to self-hosted solution (Eleventy considered). WPEngine is overpriced for the use.
3. **Archived blogs and sites** — scattered legacy content, needs sifting after 1 and 2 are done
4. **Modern web presence** — GitHub Pages sites (designinproduct.com, etc.), needs alignment with above

**Recurring costs to eliminate:** Rackspace hosting + WPEngine hosting

**Current state:** Started but paused, with cobwebs. Has an existing Claude Project. Blocked step is ezone.org repair via archive.org.

**Sequence:** Fix ezone.org → curate Enterzone → migrate mediajunkie.com off WPEngine → consolidate archives → align modern sites

**Next step:** Import the Claude Project into a Cowork session (following the VA/Archie pattern).
