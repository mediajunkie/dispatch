# Klatch style guide

Style conventions for the Klatch website, blog, and documentation. Distinct from the voice and tone guide (VOICE.md), which covers how we write — this covers the surface decisions we've settled.

---

## Capitalization

**Sentence case for headings.** Section headings, subheadings, and navigation labels use sentence case — capitalize the first word and proper nouns only.

> **Yes:** "What I'm building and why"
> **No:** "What I'm Building and Why"

**Title case for actual titles.** Post titles, product names, and named documents use title case.

> **Yes:** "Testing the Edge: Agent Experience Testing"
> **No:** "Testing the edge: agent experience testing"
