## Act II: The Systematic Awakening (12 minutes)

### Slide 5: Information Architecture saves everything
- **Your IA background is your superpower** in AI-assisted development
- Systematic verification = Research before design
- **Framework introduction**: "Check first, implement second"

### Slide 6: The Excellence Flywheel methodology
**Four Pillars for UX professionals**:
1. **Systematic Verification First** → Design audit before wireframing
2. **Test-Driven Development** → User scenario validation before UI
3. **Multi-Agent Coordination** → Strategic team deployment (researchers, designers, content)
4. **GitHub-First Tracking** → Design system documentation discipline

### Slide 7: From PM-015 chaos to 15-minute ADR migrations
- **Before**: "I think the pattern is..." (assumptions)
- **After**: "I verified the pattern is..." (evidence-based)
- **UX translation**: Research repository analysis before new studies
- **Breakthrough**: Systematic verification eliminates rework

### Slide 8: Multi-agent coordination patterns
- **Claude Code** = Senior UX strategist (complex, multi-touchpoint work)
- **Cursor** = Junior UXer (focused tasks, needs clear guardrails)
- **Deployment strategy**: Match context level to task complexity
- **UX insight**: Same patterns apply to design team coordination

### Slide 9: The 642x performance improvement story
- **Problem**: Connection-per-request architecture (classic UX: poor user flow)
- **Solution**: Systematic verification → connection pooling pattern discovery
- **Result**: 103ms → instant response times
- **UX parallel**: Information architecture fixing navigation performance

## Q&A Preparation (15 minutes)

### Anticipated Questions & Framework Answers:

**Q: "How do I convince my org to let me experiment with AI coding?"**  
**A**: Frame it as extending design impact, not changing careers. Start with systematic prototyping for user testing.

**Q: "What if I'm not technical enough for this?"**  
**A**: Your IA background gives you systematic thinking foundations. Start with verification-first approaches you already know.

**Q: "Won't this make UX jobs obsolete?"**  
**A**: The opposite. AI democratizes outputs but increases need for systematic human advocacy and strategic thinking.

**Q: "Which AI tools should I start with?"**  
**A**: Focus on tools that integrate with your existing workflow (Figma AI, content research AI) and apply systematic verification first.

**Q: "How do I learn enough about AI to be relevant?"**  
**A**: Start with systematic approaches to existing UX problems. Apply Excellence Flywheel to AI tool adoption.
