## 🛠 Troubleshooting

### Common Issues

- **"I can't find that issue"** - Use more specific references
- **"Wrong item resolved"** - Add context when multiple items exist
- **"Context lost"** - Start new session or use specific identifiers
- **"Slow response"** - Use simpler references or wait for processing

### Getting Help

- **In-App Help**: Use "help" or "?" in conversation
- **Documentation**: Reference the troubleshooting sections in each guide
- **Community**: Share tips and best practices
- **Support**: Report issues through GitHub
