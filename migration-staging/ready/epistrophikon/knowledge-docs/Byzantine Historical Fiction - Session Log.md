# Byzantine Historical Fiction Project - Session Log

## Project Overview
Working title: "The Epistrophikon" or "The Anazosis"  
Genre: Historical Fiction  
Period: 949-972 CE (focus), 900-1000 CE (context)  
Protagonist: Hraban/"Korax" - young Germanic clerk who becomes culturally Roman  
Historiographical approach: Kaldellist-influenced (avoiding "Byzantine" terminology)

## Key Decisions

### Confirmed Elements
- **Protagonist**: Hraban, born ~937-938, withered left arm
- **Core period**: 949-972 CE
- **Narrative voice**: Close third-person following Hraban
- **Central theme**: Identity transformation (Germanic → Roman)
- **Structural concept**: Possible recursive narrative with two "Epistrophika"
- **Key scene**: Calabrian prisoner interrogation (Socratic dialogue structure)

### Terminology Choices
- Medieval Roman Empire (not Byzantine)
- Romans who speak Greek (not "Greeks")
- Constantinople (not Byzantium)
- Basileus Romaion (not "Greek Emperor")

### Character Arc
1. Young interpreter (949): "The Frank"
2. Student (952-958): "The Knowledge-Lover"
3. Cultural hybrid (958-965): "The Bilingual"
4. Full Roman (965-972): "Eugene the Roman"
5. Return West (972+): "The Greek" to Germans
